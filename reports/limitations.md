# Project Limitations
- **Small Dataset**: The dataset contains only 303 instances.
- **Historical Dataset**: Data is from 1988, introducing possible distribution shift.
- **Limited Representativeness**: Narrow demographic representation.
- **Subgroup Analysis**: Subgroup sizes are too small for statistically stable fairness claims.
- **No External Validation**: The model has not been validated on external independent cohorts.
- **No Prospective Validation**: Model tested only retrospectively.
- **No Clinical Workflow Validation**: Not tested in real clinical settings.
- **Not a Diagnosis**: Model probability is statistical estimation, not a medical diagnosis.
- **SHAP Limitations**: SHAP provides feature attribution, which does not equate to biological causality.
